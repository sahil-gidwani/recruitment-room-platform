# Setup

## Repository Setup

Clone the repository and open it in your editor. The project is intentionally minimal, so the initial setup is limited to documentation, a Streamlit entrypoint, and a small dependency list.

## Environment Setup

Use a local Python environment that matches your preferred workflow. A virtual environment is recommended so dependencies stay isolated from your system Python.

## Dependency Installation

Install the project dependencies with:

```bash
pip install -r requirements.txt
```

## Running the Streamlit Application

Once dependencies are installed, start the app with:

```bash
streamlit run streamlit_app.py
```

## Development Workflow

1. Review the architecture and roadmap docs before making changes.
2. Keep changes focused on a single small task.
3. Update documentation when project structure or workflow changes.
4. Use pull requests for review and discussion before merging.
